## Title Tag Templates

### Formula: [Product Name] - [Key Feature/Benefit] | [Brand]

**Max Length:** 60 characters (including brand)

### Template 1: Premium Products
```
[Product Name] - Premium [Type] from [Origin]
```

**Examples:**
- Premium Omakase Fruit Box - Seasonal Selection
- Japanese Amaou Strawberries - Premium Fukuoka
- Korean Shine Muscat Grapes - Sweet & Seedless

### Template 2: Gift Products
```
[Product Name] - Perfect Gift for [Occasion]
```

**Examples:**
- Luxury Fruit Hamper - Perfect Corporate Gift
- Premium Fruit Box - Ideal Get Well Gift
- Seasonal Fruit Basket - Celebration Gift

### Template 3: Feature-Focused
```
[Product Name] - [Key Feature] | Ah Ho Fruit
```

**Examples:**
- Omakase Box - Same-Day Delivery | Ah Ho Fruit
- Organic Tropical Fruits - Farm Fresh | Ah Ho Fruit
- Japanese Melon - Hand-Selected | Ah Ho Fruit

### Template 4: Origin-Focused
```
[Origin] [Product Name] - [Feature] Singapore
```

**Examples:**
- Hokkaido Crown Melon - Premium Gift Singapore
- Fukuoka Strawberries - Fresh Daily Singapore
- Korean Pears - Imported Direct Singapore

### Character Count Guide:
```
50-55 chars: Optimal (leaves room for variation)
56-60 chars: Acceptable
61+ chars: Too long (will be truncated in search results)

Check length: Use Rank Math's built-in counter
```

## Meta Description Templates

### Formula: [What it is]. [Key benefit]. [Delivery/Price info].

**Max Length:** 160 characters

### Template 1: Product Description
```
[Brief product description with keyword]. [Key benefit or feature]. [Delivery info]. [Price range if applicable].
```

**Example:**
```
Curated selection of 8-10 premium seasonal fruits from Japan, Korea & local farms. Perfect for gifts or personal enjoyment. Same-day delivery available. From $88.
```
**Character count:** 157 ✓

### Template 2: Gift-Focused
```
[Product type] perfect for [occasion]. [Key features]. [Delivery]. [Call to action].
```

**Example:**
```
Premium fruit hamper perfect for corporate gifting. Elegantly packaged with ribbon. Same-day delivery across Singapore. Order now for today's delivery.
```
**Character count:** 155 ✓

### Template 3: Origin Story
```
Authentic [origin] [product] [feature]. [Quality statement]. [Delivery]. [Call to action or price].
```

**Example:**
```
Authentic Hokkaido crown melon hand-selected at peak ripeness. Premium quality guaranteed. Island-wide delivery. Limited availability - order now.
```
**Character count:** 149 ✓

### Template 4: Benefit-First
```
[Main benefit]. [Product description]. [Unique feature]. [Delivery or guarantee].
```

**Example:**
```
Impress clients with luxury fruit hampers. Premium selection of Japanese and Korean fruits. Beautiful packaging. Same-day delivery for orders before 12pm.
```
**Character count:** 158 ✓

### Character Count Guide:
```
150-155 chars: Optimal (fits all devices)
156-160 chars: Acceptable
161+ chars: Too long (will be truncated)

Include:
✓ Primary keyword naturally
✓ Key benefit or feature
✓ Call to action
✓ Delivery info or price range
```

## Image Alt Text Templates

### Formula: [Product type] [key feature] [visual description]

**Max Length:** 125 characters (7-10 words ideal)

### Template 1: Main Product Image
```
[Product name] [packaging type] with [visible fruits]
```

**Examples:**
- Premium omakase fruit box with assorted seasonal fruits
- Japanese Amaou strawberries in gift packaging
- Hokkaido crown melon in protective netting
- Corporate fruit hamper with ribbon and gift card

### Template 2: Close-Up Shot
```
Close up of [product] showing [specific feature]
```

**Examples:**
- Close up of Amaou strawberries showing perfect ripeness
- Japanese melon cross-section showing orange flesh
- Premium fruit arrangement detail shot
- Gift box contents displaying fruit selection

### Template 3: Lifestyle/Context Shot
```
[Product] [action/context] [setting]
```

**Examples:**
- Omakase fruit box on dining table setting
- Corporate gift hamper being delivered to office
- Premium fruits displayed on marble countertop
- Fruit box gift presentation at celebration

### Template 4: Packaging/Unboxing
```
[Product name] [packaging state] [packaging type]
```

**Examples:**
- Premium fruit box gift packaging with ribbon
- Omakase box unboxing showing tissue paper
- Corporate hamper in woven basket with cellophane
- Japanese melon in luxury gift box opened

### Template 5: Size Comparison
```
[Product] size comparison with [reference object]
```

**Examples:**
- Premium omakase box size comparison with hand
- Japanese melon diameter shown with measuring tape
- Fruit hamper dimensions compared to standard box
- Strawberry size comparison with coin

### Alt Text Best Practices:
```
✓ DO:
- Describe what's actually in the image
- Include product name
- Mention key visual features
- Keep under 125 characters

✗ DON'T:
- Start with "Image of..." or "Picture of..."
- Keyword stuff
- Use "click here" or "learn more"
- Leave alt text blank
```

## Internal Linking Phrases

### Link to Related Products

**Template:**
```
Pair this with our [Related Product Name] for [benefit].
```

**Examples:**
```
Pair this with our Premium Japanese Strawberries for the ultimate fruit experience.

Looking for something smaller? Try our Mini Omakase Box perfect for 2-3 people.

Upgrade to our Corporate Omakase Box for larger gatherings.

Complete your gift with our Premium Japanese Melon.
```

### Link to Categories

**Template:**
```
Browse our complete collection of [Category Name] or explore [Related Category].
```

**Examples:**
```
Browse our complete collection of Omakase Fruit Boxes or explore Corporate Gift Hampers.

Discover more Premium Japanese Fruits or view our Seasonal Selections.

Explore our full range of Corporate Gifts perfect for business clients.

See all Fruit Hampers available for delivery today.
```

### Link to Related Content

**Template:**
```
Learn more about [topic] in our [guide/blog post].
```

**Examples:**
```
Learn more about Japanese fruit grading in our Fruit Quality Guide.

Read our Complete Guide to Fruit Gifting for more ideas.

Discover How to Select the Perfect Corporate Gift in our business guide.

See our Seasonal Fruit Calendar for availability.
```

```

• [Product 1]

• [Product 3]

• [Add-on product]

• [This product] + [Complementary product]
```

## Quick Reference Checklist

### Before Publishing Any Product:

```
☐ Product name descriptive and under 70 characters
☐ URL slug clean, lowercase, hyphens only
☐ Focus keyword selected
☐ SEO title under 60 characters with keyword
☐ Meta description 150-160 characters with keyword
☐ Short description 50-100 words
☐ Long description 200+ words
☐ At least 3 product images uploaded
☐ All images have descriptive alt text
☐ All images compressed (<200KB each)
☐ SKU filled in (unique)
☐ Price set
☐ Stock status configured
☐ Primary category assigned
☐ Product tags added (5-7 max)
☐ Brand attribute set (Ah Ho Fruit)
☐ Rank Math SEO score 80+ (green)
☐ Schema markup verified (Product schema)
☐ Internal links added (2-3 links)
☐ Preview page before publishing
```
